Uma landing page moderna, elegante e responsiva desenvolvida para apresentar os produtos e a marca **Zuf's Caffè**. O projeto foi projetado com foco na experiência do usuário (UX) e alta performance, destacando a essência e o catálogo de produtos da marca de forma atrativa.

🌐 **Acesse a aplicação online:** [zufscaffe.vercel.app](https://zufscaffe.vercel.app)

---

## ✨ Funcionalidades

- **Design Responsivo:** Interface fluida e otimizada para dispositivos móveis, tablets e desktops.
- **Apresentação de Produtos:** Catálogo visual destacando as especialidades e os diferenciais do Zuf's Caffè.
- **Alta Performance:** Utilização de SSR/SSG e otimização de imagens nativas do Next.js.
- **UI/UX Moderna:** Navegação intuitiva, focada na conversão e no encantamento do cliente.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **[Next.js](https://nextjs.org/)** - Framework React utilizado para renderização e estrutura da aplicação.
- **[TypeScript](https://www.typescriptlang.org/)** - Adiciona tipagem estática, garantindo maior segurança e facilidade de manutenção no código.
- **[React](https://reactjs.org/)** - Biblioteca JavaScript para construção da interface de usuário.
- **CSS / PostCSS** - Para a estilização e responsividade dos componentes.
- **[ESLint](https://eslint.org/)** - Para padronização e linting do código.
- **[Vercel](https://vercel.com/)** - Hospedagem e deploy contínuo.
